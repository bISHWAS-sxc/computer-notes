1's and 2's Complement

1's Complement

Reciprocal of 1's to 0s and 0s to 1s is its 1's complement. For example, the 1's complement of (1011)2 is (0100)2.

2's Complement

Adding 1 to the 1's complement of a given binary number gives its 2's complement. For example, if (1010)2 is a given number, its 1's complement is (0101)2, and its 2's complement is (0101)2 + (1)2 = (0110)2.

Subtraction Using 1's and 2's Complement

Rules to remember: Let the question be X - Y.

  • Using 1's Complement: Calculate 1's complement of 'Y' and add it with 'X'. If there is an extra bit, remove it and add it to the remaining bit. If no extra bit, find the 1's complement of the result and add a negative sign.
  • Using 2's Complement: Calculate 2's complement of 'Y' and add it with 'X'. If there is an extra bit, remove it. If no extra bit, find the 2's complement of the result and add a negative sign.

Example 1

Question: Subtract (1000)2 from (111)2.

Using 1's Complement: Calculate 1's complement of (1000)2 = (0111)2. Adding (0111)2 with (111)2 gives (1110)2. The 1's complement of (1110)2 is (0001)2, and the result is -(0001)2.

Using 2's Complement: Calculate 2's complement of (1000)2 = (0111)2 + (1)2 = (1000)2. Adding (1000)2 with (111)2 gives (1111)2. The 2's complement of (1111)2 is (0001)2, and the result is -(0001)2.

Example 2

Question: Subtract (11001)2 from (11101)2.

Using 1's Complement: Calculate 1's complement of (11001)2 = (00110)2. Adding (00110)2 with (11101)2 gives (100011)2. Removing the extra bit, we get (00100)2.

Using 2's Complement: Calculate 2's complement of (11001)2 = (00111)2. Adding (00111)2 with (11101)2 gives (100100)2. Removing the extra bit, we get (00100)2.

Boolean Logic

Boolean Algebra is the algebra of logic, dealing with binary variables and logical operations. It is crucial for analyzing and designing logic circuits in modern computers, where variables can only have values 0 (False) or 1 (True). Boolean functions are formed from binary variables and operators like AND, OR, NOT.

Basic Boolean Operators

  • AND: Represented as "A.B" or "A AND B". The result is 1 only if both inputs are 1.
  • OR: Represented as "A + B" or "A OR B". The result is 1 if at least one input is 1.
  • NOT: Represented as "A'" or "NOT A". The result is the opposite of the input value.

Logic Gates

Logic gates are circuits that perform operations on binary inputs. Some basic logic gates include:

  • AND Gate: Produces output 1 only if both inputs are 1.
  • OR Gate: Produces output 1 if any input is 1.
  • NOT Gate: Inverts the input.
  • NAND Gate: Inverts the output of an AND gate.
  • NOR Gate: Inverts the output of an OR gate.
  • XOR Gate: Produces output 1 if one input is different from the other.
  • X-NOR Gate: Produces output 1 if both inputs are the same.

De-Morgan's Theorem

First Theorem: The complement of a sum equals the product of its complements: (A + B)' = A' . B'

Second Theorem: The complement of a product equals the sum of its complements: (A . B)' = A' + B'
