Introduction to the Web Technology

What is the Internet?

The Internet is the global network of networks. It is a huge collection of computers all over the world that are interconnected in various ways to form a web-net.

Components Required for Internet Connections

The basic requirements for Internet are:

  • A TCP/IP (Transmission Control Protocol/Internet Protocol) enabled computer with a web browser.
  • An account with an ISP (Internet Service Provider).
  • A telephone line.
  • A MODEM to connect to the telephone line and computer.
  • A set of computer: The capacity of the computer should be as follows:
    • Minimum 386 microprocessor chip with 16 MB RAM.
    • Color monitor with 256 color support and 640x480 resolution.
    • A hard disk of 200 MB free space.
    • Multimedia sound card and speaker.

Internet Services and Applications

World Wide Web (WWW)

WWW is a series of services that are interconnected through hypertext. It provides links in the web document. It is the most important service provided by the Internet. It was created in 1990 at the European Particle Physics Laboratory in Geneva, Switzerland to exchange and share data using the Hypertext Transfer Protocol (HTTP).

Web Browser

A web browser is a computer program that accesses web pages and displays them on the computer screen. Some popular web browsers include:

  • Internet Explorer
  • Safari
  • Mozilla Firefox
  • Opera
  • Depnet Explorer
  • Netscape Navigator
  • Mosaic

FTP - File Transfer Protocol

FTP allows users to transfer files from one computer to another over the Internet. Files may include data, graphics, text, etc.

Electronic Mail (E-mail)

E-mail allows sending and receiving messages electronically. It is the most widely used feature on the Internet, supporting text, graphics, video, animation, and audio data.

Voicemail

Voicemail is a service that answers telephone calls and records audio messages. It allows easy communication by recording and sending voice messages.

USENET – View and News

USENET is an online forum or bulletin board system (BBS) with thousands of News groups covering specific areas of the Internet.

Chat

Chat is a feature that allows real-time text communication over the Internet. Users can log on to chat rooms and connect with others. It also provides offline messaging if a friend is not online.

IRC - Internet Relay Chat

IRC is a live chat system where users can join various channels to communicate in real-time. There are major IRC servers around the world that are interconnected.

Video Conferencing

Video conferencing allows real-time video and audio communication between users across networks. It uses webcam technology for face-to-face communication.

Universal Resource Locator (URL)

URL is a unique address that identifies the location of resources on the Internet. A typical URL is structured as: 

            http                   - protocol
            www                 - host computer name
            animalplanet       - domain name
            com                  - domain type

E-Fax (Electronic Fax)

E-Fax is a service that allows users to send and receive faxes via the Internet. The message first goes to an Internet server and then to its destination, unlike traditional faxing.

TELNET – Remote Login

Telnet allows users to log in to a remote computer using a Telnet program. It enables users to work on computers across the world as if they are working locally.

Navigators – Information Servers

Information servers help users locate and retrieve data across the Internet. Popular utilities include Gopher, Archie, Hytelnet, and WWW.

Internet Terminologies

  • ISP: An Internet Service Provider gives access to the Internet.
  • Link: A clickable route to navigate to another web page.
  • Search Engine: Software that searches the web for you (e.g., Yahoo, Google, Bing).
  • Domain Name and IP Address: Domain names identify websites, while IP addresses correspond to those names.
  • DNS Server: A service that maps domain names to IP addresses.
  • Web Server: A program that stores and serves web pages to browsers.
  • USENET Newsgroups: Online forums organized by topics, like 'Sci' for science, 'Rec' for recreation.
  • POP Server: A server where incoming emails are stored until the user retrieves them.
  • SMTP Server: Server that transfers email from one system to another.
  • FTP (File Transfer Protocol): A protocol used for transferring files over the Internet.
  • HTTP (Hypertext Transfer Protocol): Protocol used for communication between web browsers and servers.
  • Proxy Server: A server that helps improve access speed and content security.

Internet Roaming

Internet Roaming is the ability to use your Internet account from anywhere in the world, provided your ISP supports it. The account details are structured like:

        krishna - user name
        @ - separates user name and domain
        wlink - ISP name
        .com - commercial organization
        .np - country code (Nepal)
